Calculator task.

Please perform 3 tasks:

  1. Create 5 functions for arithmetic operations: 'sum', 'sub', 'mul', 'div', 'rem'
    Each applies two arguments and returns result of the corresopnding arithmetical operation
  2. Create object called 'calculations' with 5 methods: '+', '-', '*', '/', '%', which performs corresponding operations
  3. Create function 'calculate' which applied 3 arguments in corresponding order: 'operation', 'a' and 'b'
    And returns result of the operation:
    • If operation is invalid it should return 'Error, unsupported operation!' message
    • If operands are not numbers (even strings '1' etc) it should return 'Error, operands should be numbers!' message

Good luck!
